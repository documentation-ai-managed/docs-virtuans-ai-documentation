## Upgrade Guide

Follow these steps to update your Virtuans AI integration to the latest version.

<Steps>
  <Step title="Check Compatibility" icon="check-circle">
    Review the changelog for breaking changes in your current version.
  </Step>
  <Step title="Update SDK" icon="download">
    Install the latest SDK package.

    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm install @virtuans/sdk@latest
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```bash
        yarn add @virtuans/sdk@latest
        ```
      </Tab>
      <Tab title="Python" icon="code">
        ```bash
        pip install virtuans-sdk --upgrade
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Reconfigure Agents" icon="settings">
    Redeploy your agents with updated configurations via the dashboard.
  </Step>
  <Step title="Test Interactions" icon="play-circle">
    Simulate customer conversations to verify new features work as expected.
  </Step>
</Steps>

## Version History

<Update label="2024-10-20" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added WhatsApp channel support for multi-channel agent deployment.
- Introduced lead scoring API endpoint to prioritize qualified prospects.

## Improvements

- Enhanced NLP accuracy by 15% for intent recognition in sales queries.
- Reduced agent response latency to `<500ms` under high load.

## Bug Fixes

- Fixed scheduling conflicts in appointment booking for overlapping time zones.
- Resolved webhook retry logic failures during network interruptions.

```javascript
// Example: New lead scoring integration
import { VirtuansSDK } from '@virtuans/sdk';

const sdk = new VirtuansSDK({ apiKey: 'your-api-key' });
const score = await sdk.leads.score({ email: 'lead@example.com', interactions: 5 });
console.log(`Lead score: ${score}/100`);
```

</Update>

<Update label="2024-10-05" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features
